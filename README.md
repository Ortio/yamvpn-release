  ## English

  ### About

  This repository hosts official APK builds of **YamVPN** — a fast and secure VPN client for Android.                      
   
  The source code is proprietary and is **not** published here. Only release binaries and their changelogs are kept in this
   repository.                                              
                                                                                                                           
  ### Download                                              

  Grab the latest build from the [**Releases**](../../releases/latest) page, or pick a specific version from the [full     
  release list](../../releases).
                                                                                                                           
  The recommended file is `app-public-universal-release.apk` — it works on all supported architectures.                    
   
  ### Install                                                                                                              
                                                            
  1. Download the APK from the release page.                                                                               
  2. On your device, allow installation from unknown sources for the browser or file manager you used.
  3. Open the downloaded APK and tap **Install**.                                                                          
                                                            
  After the first install the app updates itself automatically — manual installation is only needed for the very first     
  setup or for sideloading.                                 
                                                                                                                           
  ### Compatibility                                         

  - Android 9 (API 28) and newer                                                                                           
  - ARM64, ARMv7, x86_64

  ## Русский

  ### О репозитории

  Здесь публикуются официальные сборки APK приложения **YamVPN** — быстрого и безопасного VPN-клиента для Android.
                                                                                                                           
  Исходный код приложения проприетарный и **в этом репозитории не публикуется**. Здесь хранятся только релизные бинарники и
   описания изменений.                                                                                                     
                                                            
  ### Скачать                                                                                                              
                                                            
  Последнюю версию можно получить на странице [**Releases**](../../releases/latest), полный список версий — в [архиве 
  релизов](../../releases).                                                                                                
   
  Рекомендуемый файл — `app-public-universal-release.apk`, он работает на всех поддерживаемых архитектурах.                
                                                                                                                           
  ### Установка
                                                                                                                           
  1. Скачайте APK со страницы релиза.                       
  2. На устройстве разрешите установку из неизвестных источников для браузера или файлового менеджера, через который       
  скачивали.                                                                                                               
  3. Откройте APK и нажмите **Установить**.                                                                                
                                                                                                                           
  После первой установки приложение обновляется автоматически — ручная установка нужна только при первом запуске или при 
  ручной загрузке.
